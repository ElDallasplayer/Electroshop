Namn på trådlöst nätverk
Ange det nätverksnamn (SSID) som enheten ska associeras med.

Om enheten kommunicerar i ett ad-hoc-nätverk och det inte finns någon åtkomstpunkt, aktiverar du alternativet ”Detta är ett ad-hoc-nätverk, och det finns ingen åtkomstpunkt”. Om du väljer det här alternativet måste du också manuellt ange vilken nätverkskanal du använder. Om du inte vet vilken nätverkskanal du använder finns information i dokumentationen som följde med datorn. Leta efter informationen om kanalinställningar.
